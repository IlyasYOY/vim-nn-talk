# Write changes

В **Vim** есть команде `normal`, которая выполняет набор **Vim** операций.

Можно даже не делать `macros`, а просто набрать команду.

> [!note] Для нас все, но еще не все!
> Есть еще способы запустить код к разным местам.
>
> - g/some/norm eathing
> - `cdo`
> - `argdo`

## Example

```markdown
<!-- got -->
Утром
Пресс качат 
Бегит 
Турник 
Анжуманя

Вечером
Пресс качат 
Бегит 
Турник 
Анжуманя
```

### Solution

Команда: `normal I- [ ]`.
