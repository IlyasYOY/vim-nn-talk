# dot-operator

Получилось? Отлично, теперь можно сделать еще раз.

`.` - повторяет любой edit, что мы сделали.

> [!NOTE] Важный элемент.
> автор книги **Practical Vim** начал книгу с этого оператора.

## Example

```markdown
<!-- got -->
Утром
Пресс качат 
Бегит 
Турник 
Анжуманя

Вечером
Пресс качат 
Бегит 
Турник 
Анжуманя
```

### Solution

Команда: `I- [ ]`.
